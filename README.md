# Simple Snake Game

A classic Snake game built with Python and the `tkinter` library. No external dependencies (like `pygame`) are required!

## Features
- Smooth grid-based movement.
- Random food spawning.
- Score tracking.
- Game Over detection (walls and self-collision).

## Prerequisites
- Python 3.x installed on your system.

## How to Run
1. Navigate to the project folder.
2. Run the following command in your terminal or command prompt:
   ```bash
   python snake.py
   ```

## Controls
- **Up Arrow**: Move Up
- **Down Arrow**: Move Down
- **Left Arrow**: Move Left
- **Right Arrow**: Move Right

## Customization
You can easily modify the game settings at the top of `snake.py`:
- `SPEED`: Change the game speed (lower values are faster).
- `GAME_WIDTH` / `GAME_HEIGHT`: Change the window size.
- `SNAKE_COLOR` / `FOOD_COLOR`: Customize the colors.
